# Simulatore Caldaia MQTT
Questo progetto simula i dati di una caldaia e li invia a Home Assistant tramite MQTT.

## Configurazione
Prima di avviare il simulatore, apri il file `simulator.py` e modifica le seguenti variabili:
- `MQTT_BROKER`: L'IP della tua istanza di Home Assistant.
- `MQTT_USER`: Il tuo nome utente MQTT.
- `MQTT_PASSWORD`: La tua password MQTT.
